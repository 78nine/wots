# wots 

A very simple &quot;typeof&quot; detection

## Installation

Download node at [nodejs.org](http://nodejs.org) and install it, if you haven't already.

```sh
npm install wots --save
```


## Tests

```sh
npm install
npm test
```

## Dependencies

None
